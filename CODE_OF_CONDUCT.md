# Code of Conduct

UnaMentis is committed to providing a welcoming and inclusive environment for everyone. We expect all participants to treat each other with respect and professionalism.

The full, canonical Code of Conduct for the UnaMentis project is maintained in the main repository and applies to this client repository as well:

https://github.com/UnaMentis/unamentis/blob/main/CODE_OF_CONDUCT.md

## Reporting

If you experience or witness behavior that violates the Code of Conduct, please contact the maintainer at richard@amerman.net. All reports will be handled confidentially.

The Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
